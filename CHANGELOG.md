# Changelog

## [1.0.0] - 2025-11-04

### Added
- ✅ Kompletter Server-Setup für 20 Benutzer
- ✅ Automatische Benutzer-Erstellung mit Nextcloud-Instanzen
- ✅ WireGuard VPN Server mit Client-Configs
- ✅ Monitoring Stack (Portainer, Netdata)
- ✅ Backup-System (BorgBackup, täglich automatisch)
- ✅ Self-Healing System (automatische Container-Wiederherstellung)
- ✅ Security Hardening (Firewall, Fail2ban, SSH-Hardening)
- ✅ Dokumentation (README, Quickstart, Admin-Handbuch)

### Features
- 20 System-Benutzer (user01-user20)
- 20 Nextcloud-Instanzen (pro Benutzer)
- 20 VPN-Client-Configs (WireGuard)
- Automatische Backups (täglich um 02:00 Uhr)
- Container-Auto-Updates (Watchtower)
