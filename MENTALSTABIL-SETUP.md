# Mentalstabil - Tester Setup
## Open-Source E-Mail + KI-Video-Dashboard

### Was wir erstellen:
1. **E-Mail-Server** für mentalstabil@boksitsupport.ch
2. **Linux Dashboard** für KI-Video-Generierung
3. **Vollständig Open-Source**

---

## 1. E-Mail-Server Setup

### Option A: Mailcow (Einfach, Docker-basiert)
- Web-Interface für E-Mail-Verwaltung
- SMTP, IMAP, POP3
- Spam-Filter, Anti-Virus
- **Domain:** mentalstabil@boksitsupport.ch (oder eigene Domain)

### Option B: Mail-in-a-Box (Noob-friendly)
- Automatisches Setup
- SSL, Spam-Filter
- **Domain:** mentalstabil@boksitsupport.ch

### Option C: Postfix + Dovecot (Manuell, flexibel)
- Vollständige Kontrolle
- Komplexer Setup

---

## 2. KI-Video-Dashboard

### Option A: ComfyUI (Stable Diffusion)
- Web-Interface für Video-Generierung
- Docker-basiert
- GPU-Support (falls vorhanden)

### Option B: Stable Diffusion WebUI
- Einfaches Interface
- Video-Generierung möglich

### Option C: Custom Dashboard mit API
- Eigenes Dashboard
- API-Anbindung zu verschiedenen KI-Services

---

## Empfehlung

**E-Mail:** Mailcow (einfach, professionell)
**KI-Video:** ComfyUI (mächtig, Open-Source)
